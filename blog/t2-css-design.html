Best practices for classes and ID'sl

When and which one to use

05/31/15

Use class for multiple items with same formatting and id for one unique item on page with its own settings.

Use the class and id inside proper type HTML tags rather than div tags (eg: vs )

Remember the priority of selectors increases like so: type -> class -> id

Be mindful of use of space in selectors (or the lack of thereof)
